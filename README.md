# -
Целью работы является Разработка модели прогнозирования неблагоприятных событий у пациентов, перенесших кардиохирургическое вмешательство с использованием методов машинного обучения на основе клинико-лабораторных и инструментальных данных.
Для достижения поставленной цели решены следующие задачи: реализована предобработка клинико-анамнестических данных; построены, оптимизированы и сопоставлены шесть прогнозных моделей (от линейных до градиентного бустинга и ядерных методов); осуществлен сравнительный анализ их устойчивости к переобучению; рассчитана метрика важности признаков.
Полученные результаты: Определен оптимальный алгоритм для клинического внедрения — Random Forest, продемонстрировавший высокую точность ранжирования на независимом тесте (ROC-AUC = 0.8494) при высокой стабильности архитектуры. Выявлены четыре наиболее значимых факторов риска, аккумулирующих около 55% предсказательной силы: «Объем операции» (17.311%), «Класс стенокардии» (13.862%), «Функциональный класс NYHA» (12.905%), «Легочная гипертензия» (10.570%).
На основании работы рекомендовано: в качестве ядра систем поддержки принятия врачебных решений для прецизионной оценки госпитальной летальности рекомендуется использовать ПМ IV (Random Forest). Для задач первичного экспресс-мониторинга и выделения максимального пула пациентов высокого риска рекомендуется интеграция алгоритма ПМ I (Logistic Regression).
